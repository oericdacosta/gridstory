# ML Module

This module contains machine learning algorithms for event detection and analysis.

## Planned Components

- **degradation.py**: Tire degradation detection using Ruptures (Change Point Detection)
- **anomaly.py**: Anomaly detection using Isolation Forest (driver errors, outliers)
- **clustering.py**: Stint clustering using K-Means/DBSCAN
- **events.py**: Event detection orchestrator (undercuts, tire drops, etc.)
- **orchestrator.py**: Complete ML pipeline
